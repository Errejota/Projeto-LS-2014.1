Estrutura:

_img/ - possui as imagens utilizadas no site
_js/ - possui os códigos usados no index(códigos de validação foram mantidos no mesmo arquivo do formulário
_pg/ - todas as páginas das abas
_slide/ - possui as imagens utilizadas no slide do plano de fundo
index.html - pagina inicial.

--------------------------------------------------------------------

Configuração: O site deve ser aberto no navegador Mozila Firefox

--------------------------------------------------------------------

Funcionalidade: Se trata de um site sobre o jogo League of Legends
no qual voce pode se informar sobre as noticias mais recentes
sobre o jogo, se cadastrar, entrar em contato com nossa equipe
e ficar por dentro dos lançamentos de novos campeões.
